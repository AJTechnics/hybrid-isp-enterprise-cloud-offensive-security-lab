# Oakcross NPCs

Theme: farming + road checkpoint town.

## Ambient NPC behavior
- Walk local routes near tavern, market, and road checkpoint
- Use short rotating dialogue for world flavor

## Functional roster
- Maren (Town Crier): event board, weekly announcements
- Elira (Innkeeper): food, rumors, starter quests
- Bram (Blacksmith): tools, repair flow, ore quests
- Captain Rowan: patrol and defense quests
- Sella (Market Clerk): rotating stock and event goods
- Fen (Scout): exploration hints and map guidance

## Progression behavior
- Dialogue and quest branches unlock by reputation tier
- Some NPC lines change by season and weekly event
- NPC shop stock scales with reputation where relevant
