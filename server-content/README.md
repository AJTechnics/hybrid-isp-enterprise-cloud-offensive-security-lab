# Living World Content Blueprint

Goal: evolve the current Paper server into a living world while preserving all existing builds.

Core rules:
- No world reset
- No terrain regeneration
- No replacing existing builds
- Add only NPCs, shops, quests, decorations, and event systems

Plugin stack target:
- Citizens
- NotQuests
- Shopkeepers
- NewSeasons
- EssentialsX

Optional later:
- BetonQuest

Rollout phases:
1. Install/validate plugins and backup world state
2. Build Oakcross MVP (NPCs, shops, starter quests)
3. Expand to nearby areas and connect content
4. Add full seasonal + weekly event progression
