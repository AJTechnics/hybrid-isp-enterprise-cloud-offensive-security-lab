# Road Patrol

Purpose: reinforce village perimeter routes using existing roads and checkpoint builds.

## Active window
- Weekly rotation slot: week 2
- Duration: 7 days

## NPC hooks
- Captain Rowan Pike: patrol assignment and completion check.
- Fen: optional scout route extension hint.

## Objectives
- Patrol gate to east bridge and return.
- Clear minor hostile pressure on road segment.
- Report back to Captain Rowan.

## Rewards
- Base: currency and guard reputation.
- Bonus: patrol commendation token for repeat completion.

## Cleanup
- Revert event dialogue to normal patrol contracts.
- Disable temporary patrol bonus modifier.
- No terrain edits or structure replacement.
